# TryHackMe Walkthroughs

This directory contains walkthroughs and notes for TryHackMe rooms organized by difficulty level.

## Directory Structure

- **beginner/**: Beginner level rooms

## Usage

Each room has its own directory with:
- `README.md`: Overview of the room
- `notes.md`: Detailed walkthrough and analysis
- `commands.md`: Useful commands used during the room
- `screenshots/`: Screenshots and visual documentation
